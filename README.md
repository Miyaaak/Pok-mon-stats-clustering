# Pokemon-stats-clustering
Clustering analysis of Pokémon base stats using K-means and PCA.
# Pokémon Base Stats Clustering Analysis

本プロジェクトでは、ポケモンの種族値データを題材にK-means クラスタリングと主成分分析（PCA）を用いた特徴量分析を行った。  
ゲームデータは特徴量が明確で、教師なし学習の実践に適しているため、  
クラスタリング手法の理解と可視化を目的として分析を実施した。

---

## 1. 目的
- 種族値（HP / 攻撃 / 防御 / 特殊 / 素早さ）の構造を理解する  
- K-means によるクラスタリングを実践する  
- PCA によって高次元データを可視化し、特徴の違いを解釈する  
- クラスタ数の選択理由を説明し、分析の妥当性を示す  

---

## 2. 使用データ
- 対象：初代ポケモンの種族値データ（151種類）  
- 特徴量：HP, こうげき, ぼうぎょ, とくしゅ, すばやさ
- 前処理：StandardScaler による標準化  

---

## 3. 手法
- StandardScalerによる標準化  
- K-meansクラスタリング  
- PCA（主成分分析）による2次元可視化  
- **Plotly** によるインタラクティブな散布図生成  

---

## 4. クラスタ数の選択理由（k=4）
エルボー法では k=2 付近で SSE の減少が緩やかになり、形式的には 2 が候補となる。  
しかし、k=2 では「強い／弱い」といった大まかな二分にしかならず、  
ポケモンの特徴や役割を十分に捉えられない。

一方で **k=4** を採用すると、PCA による可視化でクラスタ間の分離が明確になり、

- 高速アタッカー  
- 耐久寄り  
- バランス型  
- 火力特化型  

といった役割に対応した解釈可能な分類が得られた。  
そのため、本分析では解釈可能性と分析目的を優先し、k=4 を採用した。

---

## 5. 結果（PCA 可視化）
PCA によって 6 次元の種族値を 2 次元に圧縮し、クラスタリング結果を可視化した。  
Plotly を用いることで、各ポケモン名をホバー表示でき、直感的な分析が可能となった。

## 6. 考察
クラスタリングの結果、ポケモンは種族値の傾向に応じて複数の特徴的なグループに分かれた。  
第1主成分は総合的な種族値の高さを表し、第2主成分は「素早さと耐久のトレードオフ」を示す傾向が見られた。  
これにより、種族値の構造がクラスタリングに反映されていることが確認できた。
